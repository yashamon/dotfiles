Privacy Policy

This extension does not collect any user or usage information whatsoever, and does not serve any ads.

The Tex for Gmail extension is built on top of Inbox SDK (also used by Dropbox for Gmail, Streak CRM for Gmail and tens of other popular extensions). When using Rich Math mode, Tex for Gmail replaces mathematical formulas (typeset in TeX and properly enclosed in $) by images generated by Codecogs and WordPress. Therefore, Codecogs and WordPress receive the text of the formulas in order to create equivalent images.

Miscellaneous
