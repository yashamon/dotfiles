Bing Wallpaper App for Chromebooks

Thanks for installing this ChromeOS app. Please, note that this app works only on the Chromebooks and other ChromeOS devices. It Doesn't work on Windows, Linux or macOS.

This app automatically sets Bing.com's Image of the Day as the desktop wallpaper on ChromeOS devices. Moreover, it checks for new wallpaper periodically in background, and updates system wallpaper when a new wallpaper is available.

Bing wallpapers are restricted to wallpaper use only.
