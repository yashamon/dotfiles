Obtaining a private network address

In order to connect over the same local network, you must obtain the remote computer's private network address. This may be:

  • A host name, for example iMac or beethoven.acme.com. Examine Control Panel > System (Windows) or System Preferences > Sharing > Screen Sharing (Mac).
  • An IPv4 address, for example 192.168.2.187. Examine the VNC Server dialog, or run a command such as ifconfig (Mac and Linux) or ipconfig (Windows).

By default, VNC Server listens on the registered port for VNC, 5900. However, it is possible for this port to change, in which case you must qualify the network address with the new port number when you establish a connection.

For more information, please read the online guide.
