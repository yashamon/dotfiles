Setting up the computer

In order for VNC Viewer to connect, VNC-compatible software must be installed and running on the remote computer you want to control.

RealVNC recommends VNC 5 for robust and responsive connections and, depending on how it is licensed, for ultra-secure and high-performance connections too. Free and trial licenses are available.

For more information on obtaining and setting up VNC on a wide range of computers, please read the online guide.
