Setting up the network

If you are connecting over the Internet, the remote computer is very likely to be protected by at least one firewall and router. If this is the case, you must:

  1. Configure its firewall to allow network communications through to a particular port (5900 by default).
  2. Configure its router to forward network communications through to the same port. We recommend you assign a static IP address to the remote computer, for example 192.168.0.1.
  3. Obtain its public network address, which is that of the router, and use this to connect. We recommend that if your ISP changes your router's address periodically, you apply for a host name (using a service such as www.noip.com).

For more information on these operations, please read the online guide.
