Setting up a same network connection

You can use VNC Viewer to control a computer connected to the same local network as your Chrome-enabled device, for example a LAN or VPN. This is typically useful to connect two devices at home or in a small office environment.

Before you can establish a same network connection, you must:

  1. Set up the computer you want to control
  2. Obtain its private network address

You must perform these operations before you can establish a connection for the first time.
