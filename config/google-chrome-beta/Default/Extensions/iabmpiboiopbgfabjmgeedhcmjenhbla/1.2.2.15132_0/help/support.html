Getting help or reporting a problem

In addition to these pages, we provide connection information in an online guide.

If you experience a problem, you can contact Technical Support. We regret that we cannot offer telephone support at this time. Please be sure to include in the body of the enquiry:

  • Whether you are connecting over the same local network or the Internet.
  • The version number of the VNC Viewer app.
  • What type of device you have (for example, a Windows 8 PC or a Samsung Chromebook).
  • Information about the remote computer you are trying to connect to: platform, OS, and version number of VNC.

    • Note: If you are connecting to third-party VNC-compatible technology, please be sure to say exactly what it is, though note the Support team cannot help you configure it.
