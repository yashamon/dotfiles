Obtaining a public network address

In order to connect over the Internet, you must obtain the remote computer's public network address. This is typically the IPv4 address of a router protecting the remote computer, for example 212.44.6.82. To discover it, navigate to www.whatismyip.com from a web browser on the remote computer.

By default, VNC Server listens on the registered port for VNC, 5900. However, it is possible for this port to change, in which case you must qualify the network address with the new port number when you establish a connection.

For more information, please read the online guide.
