Known Issues
------------

* Sometimes mouse events are lost after entering fullscreen on OS X.  Switching tabs will fix the issue.
* We have seen some issues where corrputed local storage can prevent future connections.  Delete your local storage and reconnect.
* Infrequent black screen seen after remote desktop resizes.
* Help links point to an internal RealVNC server.
* The connection quality cannot be changed once connected when using a 3rd party VNC server.
* Connections to Tight VNC sometimes use the wrong colour palette.
