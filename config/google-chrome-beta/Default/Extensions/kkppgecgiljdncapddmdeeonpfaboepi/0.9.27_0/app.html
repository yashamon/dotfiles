Bing Wallpaper App for Chromebooks

This app works only on the Chromebooks and other ChromeOS devices. Doesn't work on Windows or macOS.

It automatically sets Bing.com's Image of the day as the desktop wallpaper on Chromebooks and other ChromeOS devices. Checks periodically for new wallpapers in background. Please note that the Bing wallpapers are restricted to wallpaper use only.

App by BrowserNative. Have any feedback or suggestion? Visit our home page, or e-mail at browsernative.apps@gmail.com.

Check our other apps and extensions available for Chrome and other browsers.

If you have a Windows or macOS device, please try our this extension which sends notifications for new Bing wallpapers, and provides an easy option to download them.
